#git 的常用指令

**1.git clone 地址 ———— 下载远程仓库的东西**

**2.cd 文件夹名 ———— 打开文件夹**

**3.ls ———— 显示当前文件夹的东西**

**4.git status ———— 显示当前文件夹的改动情况**

**5.git add . ———— 把工作区的东西推到暂存区**

**6.git commit-m"本次修改的理由" ———— 把暂存区的东西推送到本地仓库**

**7.git pull origin master ———— 把远程仓库的东西更新到本地仓库**

**8.git push origin master ———— 把本地仓库的东西推送到远程仓库**
